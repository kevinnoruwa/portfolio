Flixstre

View Project View code

Flixstre is a movie website built with Next.js and React. Flixstre implements the movie database API that is request to through React Hooks. One of my favorite features on the website is the watch list feature which allows you to save movies and visit them later, this is done by JavaScripts local server which was one of the last implementation of this project.
